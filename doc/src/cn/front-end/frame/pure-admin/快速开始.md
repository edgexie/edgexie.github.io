## 快速开始

启动项目

```shell
# clone 项目
git clone git@github.com:pure-admin/pure-admin-thin.git

cd pure-admin-thin

pnpm install

pnpm dev
```

技术栈：vue3 vite ts pinia tailwindcss element-plus vite-plugin-fake-server\(mock\) [iconify](https://icones.js.org/)\(图标框架\)
