组件 ​

在单文件组件中，子组件推荐PascalCase的标签名。

在 DOM 中直接书写，推荐kebab-case的标签名

父子组件通信 ​

子组件定义 props defineProps

vue
<!-- 运行时声明 -->
<script setup lang="ts">
defineProps({
  title: {
    type: String,
    required: true,
  },
});
</script>

<!-- 基于类型的声明 -->
<script setup lang="ts">
const props = defineProps<{
  foo: string;
  bar?: number;
}>();
</script>

透传 attribute ​

vue3 移除了 $listeners，事件在$attrs 中以字段onClick存在。 属性会自动继承给子组件的最外层 dom，如不想自动继承 需使用defineOptions( {inheritAttrs: false} )

vue
<MyComponent first-name="123" second-name="234" @click="handleClick"/>

<script setup>
defineOptions({
  inheritAttrs: false
})
</script>

<template>
  <div>
    <span v-bind="$attrs">
  </div>
</template>
