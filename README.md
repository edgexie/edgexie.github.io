这个项目是用 [lerna](https://github.com/lerna/lerna) 组织的

文档相关代码在 doc 目录下

## 启动项目

```shell
git clone git@github.com:edgexie/lerna-starter.git
cd lerna-starter
pnpm install
pnpm run docs:dev
```

项目使用 git actions 自动部署，[参考](https://github.com/edgexie/lerna-starter/blob/master/.github/workflows/blank.yml)
